## 四、阶段 2：核心功能

### 4.1 添加长期记忆

```python
# src/agent/main.py
from deepagents.backends import CompositeBackend, StateBackend, StoreBackend
from langgraph.store.sqlite import SqliteStore
from langgraph.checkpoint.memory import MemorySaver

def create_mining_agent_with_memory():
    # 配置持久化存储
    store = SqliteStore.from_path("./data/memories.db")
    store.setup()

    checkpointer = MemorySaver()

    def make_backend(runtime):
        return CompositeBackend(
            default=StateBackend(runtime),
            routes={"/memories/": StoreBackend(runtime)}
        )

    agent = create_deep_agent(
        model=llm,
        store=store,
        backend=make_backend,
        checkpointer=checkpointer,
        tools=tools,
        system_prompt=MINING_SYSTEM_PROMPT,
    )

    return agent
```

### 5.2 主进程代码

```javascript
// main.js
const { app, BrowserWindow, ipcMain } = require('electron');
const { spawn } = require('child_process');
const path = require('path');

let mainWindow;
let pythonProcess;

// 嵌入式 Python 路径
const pythonPath = path.join(
    process.resourcesPath,
    'python',
    'python.exe'
);

function createWindow() {
    mainWindow = new BrowserWindow({
        width: 1200,
        height: 800,
        webPreferences: {
            preload: path.join(__dirname, 'preload.js'),
            nodeIntegration: false,
            contextIsolation: true
        }
    });

    mainWindow.loadFile('renderer/index.html');
}

function startPythonAgent() {
    const agentScript = path.join(
        process.resourcesPath,
        'agent',
        'server.py'
    );

    pythonProcess = spawn(pythonPath, [agentScript]);

    pythonProcess.stdout.on('data', (data) => {
        console.log(`Agent: ${data}`);
    });

    pythonProcess.stderr.on('data', (data) => {
        console.error(`Agent Error: ${data}`);
    });
}

// IPC 通信
ipcMain.handle('chat', async (event, message) => {
    // 与 Python Agent 通信
    // 可以通过 HTTP、WebSocket 或 stdio
});

ipcMain.handle('resume', async (event, userInput) => {
    // 恢复中断的 Agent
});

app.whenReady().then(() => {
    startPythonAgent();
    createWindow();
});

app.on('window-all-closed', () => {
    if (pythonProcess) {
        pythonProcess.kill();
    }
    app.quit();
});
```

### 5.3 Python Agent 服务

```python
# agent/server.py
from flask import Flask, request, jsonify
from src.agent.main import create_mining_agent_with_memory

app = Flask(__name__)
agent = create_mining_agent_with_memory()
threads = {}  # 存储会话状态

@app.route('/chat', methods=['POST'])
def chat():
    data = request.json
    thread_id = data.get('thread_id')
    message = data.get('message')

    config = {"configurable": {"thread_id": thread_id}}

    response = agent.invoke({
        "messages": [{"role": "user", "content": message}]
    }, config=config)

    return jsonify(response)

@app.route('/resume', methods=['POST'])
def resume():
    data = request.json
    thread_id = data.get('thread_id')
    user_input = data.get('input')

    config = {"configurable": {"thread_id": thread_id}}

    from langgraph.types import Command
    response = agent.invoke(
        Command(resume=user_input),
        config=config
    )

    return jsonify(response)

if __name__ == '__main__':
    app.run(port=5000)
```

## 八、常见问题

### Q1: 百炼 API 调用失败

检查：
- API Key 是否正确
- 网络是否通畅
- 额度是否用完

### Q2: Tavily 搜索无结果

检查：
- API Key 是否正确
- 关键词是否太长
- 是否需要配置代理

### Q3: IRR 计算结果不对

检查：
- 现金流符号是否正确（投资为负）
- 单位是否统一
- 计算公式是否正确

### Q4: 记忆没有持久化

检查：
- SQLite 文件路径是否正确
- 是否调用了 `store.setup()`
- `/memories/` 路由是否配置

### Q5: Electron 打包后 Python 无法运行

检查：
- 嵌入式 Python 路径是否正确
- `python312._pth` 是否配置正确
- 依赖库是否都打包进去
- `frontend/resources/python/python.exe` 与 `frontend/release/win-unpacked/resources/python/python.exe` 是否同时存在
